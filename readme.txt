Adds new custom post types Courses, Modules, and Lessons. Allows for basic management and linking of them.

== Description ==
Allows for the creation of web courses with the addition of three new custom post types, Courses, Modules, and Lessons. Allows for the linking of Lessons to one or more Modules, and Modules to one or more Course. Everything is contained within one new menu item.

== Installation ==
Download and place inside your plugins folder inside your WordPress directory. Then activate from the plugins menu on your site.
